# -CODSOFT_PP_02--Password-Generator
The Password Generator project is a secure and customizable tool that generates strong and random passwords for users. It allows users to choose password length and include options such as uppercase letters, lowercase letters, numbers, and special characters. The goal of this project is to promote better security practices by encouraging the use of complex passwords instead of weak or repeated ones. This application showcases the use of randomization, string manipulation, and user input handling. It is useful for real-world scenarios where password security is important and serves as a practical example of utility-based software development.
